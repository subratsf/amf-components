AMF graph API documentation

A set of components to render and edit documentation for API projects.

Documentation viewers

Api navigation
The main navigation view.
Api documentation
The main element that renders the documentation from the current AMF model and navigation selection.
Api documentation (partial model)
The main documentation element with a partial AMF model support.
Api resource (an endpoint)
A documentation viewer for an API endpoint. Works with HTTP APIs.
Api channel
A documentation viewer for an API channel. Works with Async APIs.
Api Operation
A documentation viewer for an API operation. Works with both HTTP and Async APIs.
Api summary
A start page with API general info
Api server selector
An element to render a list of API servers
Api documentation
A documentation viewer for an API Documentation.
Api schema documentation
A documentation viewer for an schema (data type).
Api security schema documentation
A documentation viewer for an security schema.
Api annotation
An annotation render element.
Api payload
A documentation for a payload (request and response).

Request editor

API request editor
An HTTP editor UI powered by the AMF model.
API request
An HTTP editor UI powered by the AMF model with the response view.

Experience

API Console
A dynamically created documentation page for an API.
API Editor
A text editor for RAML and OAS.
